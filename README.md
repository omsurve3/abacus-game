This is a single-page interactive abacus game built using React, HTML, CSS, and JavaScript.

## Features
- 5-rod abacus (Units to Ten-Thousands)
- Random target number generation
- Real-time value calculation
- Carry-over logic between rods
- Visual and audio feedback on success
- Hint mode to guide learners
- Child-friendly UI with animations

## How to Run
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`

## Tech Stack
- React (JavaScript)
- HTML (JSX)
- CSS (Custom styles)
